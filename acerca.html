ACERCA DE

Este sitio web explora el movimiento de un personaje en videojuegos utilizando a Solaris como ejemplo visual. A través de diferentes secciones se presentan acciones como caminar, saltar y cambiar entre estados, con el objetivo de mostrar cómo funcionan las animaciones y transiciones dentro de un entorno interactivo.
